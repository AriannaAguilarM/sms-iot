# SMS-IoT: Sistema Inteligente de Sueño IoT

## 📌 Descripción del problema
Muchas personas no tienen un control adecuado sobre la calidad de su sueño, lo que puede provocar fatiga, bajo rendimiento y problemas de salud. Este proyecto busca monitorear variables relacionadas con el sueño mediante sensores IoT.

## 🎯 Objetivo general
Desarrollar un sistema IoT capaz de monitorear y analizar la calidad del sueño del usuario en tiempo real, utilizando sensores y almacenamiento en la nube.

## 🧰 Tecnologías utilizadas
- ESP32
- Firebase Realtime Database
- HTML / CSS / JavaScript
- Git y GitHub
- (Opcional) CodeIgniter 4 para backend

## 📡 Sensores y actuadores empleados
- Sensor de movimiento (PIR o acelerómetro)
- Sensor de temperatura y humedad (DHT11 o BME280)
- Sensor de ritmo cardíaco (MAX30102) *(si aplica)*
- Buzzer o LED para alertas
